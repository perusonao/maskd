## 基本原則

- **足すのではなく、削る・後ろにずらす・仕上げる。** 新しいルール・属性・行動は追加しない。
- **初回体験は「読ませてから遊ばせる」ではなく「遊ばせながら必要なことだけ教える」**（見せるタイミングを遅らせる）。
- **段階システム（初級→中級→上級／★／ライバル／仮面あて／看破・バレ）は在庫として維持**し、初見に一度に当てない。登場を解放に同期させる。

### v1.0 完成の定義（Definition of Done）
1. 初訪問ユーザーが詰まらず最後まで1戦できる
2. JSエラー0・主要画面で見切れなし
3. **計測がPhase3の問いに答えられる**（A4）
4. タイトル/アイコン/キャッチコピー/スクショが揃う
5. `v1.0.0` をGitタグ＝**フリーズ宣言**（以後 main は凍結、改修はデータ後）

## メモ

- バランス検証はシミュレータ（本物のCPUロジック移植）で。ランダムbotではなく実ロジックで測る。
- 変更は毎回 Playwright headless で JSエラー0・通し完走・見切れなしを確認してからデプロイ。
- デプロイは main へ push → GitHub Pages（pages build and deployment）。
- バージョンは `GAME_VERSION` と `CHANGELOG` を必ず両方更新（詳細な引き継ぎは HANDOFF.md）。
- README は既に最新内容（テーマ「仮面騎士の戦い」・現行機能）へ更新済み。A5の残タスクはスクショ・30秒キャプチャ・X投稿文。
