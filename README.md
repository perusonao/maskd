# MASKD

仮面舞踏会をテーマにした、2人対戦（プレイヤー vs CPU）の心理戦・推理カードゲーム。

## 概要

- 全8ラウンド。毎ラウンド、親（ディーラー）と子（チャイルド）の非対称な役割が交代する
- 親が先にカードを伏せて置き、子はそれを見てから置く。親だけが「決戦・撤退・入替」を選べる
- Sun / Dark / Moon / Kaos / Air の5属性 × 0〜5 の6数字＝**全30枚**の固定デッキを2人で15枚ずつ分配。強力な低Kaos（0・1）は両者に1枚ずつ配って配札運を抑える
- 初期の実働手札**5枚**は**ランダム**（毎回ちがう手札で展開が変化）。残りは山札から補充。自分の山札の残り枚数・次に引く札は画面に表示され、タップで中身を確認できる
- カードは伏せて出し、撤退（リトリート）や奇襲（アンブッシュ）、親の特権「入替」といった駆け引きが発生する

## 勝敗ルール（要点）

- **Air** が絡む → 必ず引き分け。ただし **Kaos は数字問わず Air に勝つ**
- **Kaos** が絡む（Airなし）→ 数字が大きい方の勝ち。同値は Kaos 側が敗北
- 特殊ルール：Kaos**0・1** は全属性の**5に勝つ**／Kaos**5** は全属性の**5に打ち勝つ**が**1には負ける**
- 同属性同士は数字勝負。異属性（Sun/Dark/Moon）は Moon→Dark→Sun→Moon の3すくみ
- **終盤決戦**：最終2ラウンド（第7・8R）の決戦は得点が2倍

## 得点・戦略カウンター

- 勝った側は相手の仮面の数字ぶん得点を得る（初期値0、負けても減らない）
- 各自「戦略カウンター」（初期値5）を持ち、撤退-1／入替-2／奇襲-2、決戦勝利+1・Air使用+1（上限5）で増減する
- 最終得点 ＝ 獲得点 ＋ 残り戦略カウンター ＋ 残り手札ボーナス（未使用カードの数字、Airは2倍）

## 遊び方

このリポジトリを GitHub Pages で公開すると、ブラウザだけでそのまま対戦プレイできます。
スマホでは「ホーム画面に追加」することでアプリのように起動できます。

ルールの全文・変更履歴はアプリ内（ヘッダーの「MASKD」タップ）から確認できます。

## 開発ステータス

プロトタイプ段階。ルールバランスは、本物のCPU意思決定ロジックを移植したシミュレーションで検証しながら継続調整中。
