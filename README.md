# MASKD

> **仮面舞踏会を舞台にした、心理戦・推理カードゲーム。**
> 相手の伏せた仮面（属性）を推理し、親と子の駆け引きで得点を競う ― 2人対戦（プレイヤー vs CPU）。

## 概要

- 全8ラウンド。毎ラウンド、親（ディーラー）と子（チャイルド）の非対称な役割が交代する
- 親が先にカードを伏せて置き、子はそれを見てから置く。親だけが「決戦・撤退・入替」を選べる
- Sun / Dark / Moon / Kaos / Air の5属性 × 0〜5 の6数字＝**全30枚**の固定デッキを2人で15枚ずつ分配。強力な低Kaos（0・1）は両者に1枚ずつ配って配札運を抑える
- 初期の実働手札**5枚**は**ランダム**（毎回ちがう手札で展開が変化）。残りは山札から補充。自分の山札の残り枚数・次に引く札は画面に表示され、タップで中身を確認できる
- カードは伏せて出し、撤退（リトリート）や奇襲（アンブッシュ）、親の特権「入替」といった駆け引きが発生する
- カード表示は**初心者向け**（属性色・大きなじゃんけん記号・数字・属性名）と**上級者向け**（リッチな仮面イラスト）を設定で切り替え可能（初期値は初心者向け）

## 勝敗ルール（要点）

- **Air** が絡む → 必ず引き分け。ただし **Kaos は数字問わず Air に勝つ**
- **Kaos** が絡む（Airなし）→ 数字が大きい方の勝ち。同値は Kaos 側が敗北
- 特殊ルール：Kaos**0・1** は全属性の**5に勝つ**／Kaos**5** は全属性の**5に打ち勝つ**が**1には負ける**
- 同属性同士は数字勝負。異属性（Moon/Dark/Sun）は**3すくみ＝じゃんけん**：月✊＞闇✌＞太陽✋＞月（カード右上に記号を表示）
- **終盤決戦**：最終2ラウンド（第7・8R）の決戦は得点が2倍

## 得点・戦略カウンター

- 勝った側は相手の仮面の数字ぶん得点を得る（初期値0、負けても減らない）
- 各自「戦略カウンター」（初期値5）を持ち、撤退-1／入替-2／奇襲-2、決戦勝利+1・Air使用+1（上限5）で増減する
- 最終得点 ＝ 獲得点 ＋ 残り戦略カウンター ＋ 残り手札ボーナス（**手札に残った非Air札のうち数字の大きい上位2枚**を合計、**Airは数字に関係なく1枚2点**。山札は対象外）

## 主な機能

- **ミッション形式チュートリアル**：実際にカードを出して操作とルールを覚える。基本11問（操作・じゃんけん・数字・撤退・Kaos・Air・入替・奇襲・終盤2倍・Kaos弱点）＋上級4問（KaosはAirを斬る・同値の落とし穴・Airの過信・三択の見極め）。初回起動時にウェルカム画面で案内
- **CPU対戦**：3つの性格（積極的＝剛力仮面／慎重＝鉄壁仮面／トリッキー＝技巧仮面）× 3段階の強さ（弱い／普通／強い）。思考は「親の1手読み」等を実装し、本物の判定を移植したシミュレータで勝率を検証済み
- **実績（アチーブメント）18種**：初勝利・連勝・完全試合・大逆転・混沌の一撃（Kaos0/1で5を食う）・風の使い手・上級突破・覇者 など。試合終了やミッション修了で自動解除し、進捗を一覧表示
- **やり込み統計**：通算勝率・最高/現在連勝・今回のMVPカード・通算MVP、各ラウンドの内訳や残り手札の内訳も確認可能
- **演出・効果音**：カードの3Dめくり・決着のインパクト演出・ハプティクス（対応端末）＋WebAudio合成の効果音（フッターでON/OFF）。`prefers-reduced-motion` にも配慮
- **2つのカード表示**：初心者向け（属性色・じゃんけん記号・数字・属性名）／上級者向け（リッチな仮面イラスト）を切替
- **PWA対応**：スマホの「ホーム画面に追加」でアプリのように起動

## 遊び方

このリポジトリを GitHub Pages で公開すると、ブラウザだけでそのまま対戦プレイできます。
スマホでは「ホーム画面に追加」することでアプリのように起動できます。

ルールの全文・変更履歴はアプリ内（ヘッダーの「MASKD」タップ）から確認できます。
