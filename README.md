# MASKD

仮面舞踏会をテーマにした、2人対戦の心理戦・ブラフ系カードゲーム。

## 概要

- 毎ラウンド、親(ディーラー)と子(チャイルド)の非対称な役割を交代
- カードは伏せて配置し、撤退(リトリート)や奇襲(アンブッシュ)といった駆け引きが発生
- Sun / Dark / Moon / Kaos / Air の5属性、0〜5の数値で構成された全25枚の固定デッキ

## 遊び方

このリポジトリを GitHub Pages で公開すると、ブラウザだけでそのまま対戦プレイできます。
スマホでは「ホーム画面に追加」することでアプリのように起動できます。

## 開発ステータス

プロトタイプ段階。ルールバランスはシミュレーションで検証しながら継続調整中。
